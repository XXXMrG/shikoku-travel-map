# 四国旅行地图｜晓倩专属攻略

一个定制化的四国旅行决策网站：用交互式示意地图组织 13 天疗愈主线、四县区域攻略、瀬户内跳岛分支、住宿与交通取舍规则。

## 内容来源

- 四国疗愈旅行攻略 PDF：2026.09.25–10.07，13 天 12 晚主线。
- 旧本地四国专页与四条路线总览。
- 小红书截图经验提炼：栗林公园、直岛/丰岛、小豆岛、高屋神社、高知西部、交通票券与住宿策略。

## 本地运行

```bash
npm install --legacy-peer-deps
npm run dev
```

默认开发地址：<http://127.0.0.1:5184/>

## 构建

```bash
npm run build
npm run preview
```

默认 preview 地址：<http://127.0.0.1:4184/>

## 当前功能

- 单独地图页：不再混入长篇路线、图库和来源说明。
- 真实四国县域轮廓地图：香川、德岛、高知、爱媛使用 GeoJSON 轮廓生成。
- 10 个地点/分支点：高松、小豆岛、琴平、祖谷/大步危、高知、四万十川、大洲/内子、松山/道后、直岛/丰岛、高屋神社。
- 地图点击切换区域攻略与地点攻略。
- “想保留 / 待定 / 可删”本地浏览器决策标记。
- 13 天主线压缩为底部简表，瀬户内跳岛、温泉强化、高知西部只作为底部提醒。
- 桌面与移动端响应式布局。
