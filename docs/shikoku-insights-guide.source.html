这版怎么用

  • 想要不累：直接走 PDF 13 天主线。

这次图片里真正有用的信息

截图不是简单加景点，而是改变了路线策略：高松不只作为入口，瀬户内岛屿值得单独成线；跳岛住宿不一定继续住高松；四万十/高知西部适合当主线后段，不要被城市温泉带跑。

1｜高松前段可更细

栗林公园值得预留半天：掬月亭、水榭周边、飞来峰、偃月桥、芙蓉峰都适合慢拍。茶席约 9:00–16:30，花园亭朝粥约 7:00–10:00 且需提前电话预约，游船约 9:00–16:00。

2｜直岛要租电助力车

截图反复提到直岛巴士间隔偏长。直岛若想舒服看地中美术馆、南寺、李禹焕/Benesse、黄南瓜，本质上要提前预约美术馆并尽量租电动助力车，不要临场靠巴士硬串。

直岛James Turrell提前预约

3｜跳岛别死守高松住宿

国人假期 + 日本假期 + 瀬户内艺术祭会推高高松房价。宇野到直岛/丰岛更近、船班也多，适合作为跳岛基地；缺点是餐饮和夜生活比高松村，JR 去冈山倒方便。

宇野Sadou Inn艺术祭避坑

4｜小豆岛要做减法

小豆岛很大，1 天不要贪全岛。可抓妖怪美术馆、迷路街道、橄榄公园、酱油相关街区；24 瞳映画村不一定能到，天使之路评价一般，潮汐不顺就别硬追。

小豆岛酱油之乡橄榄公园

5｜高屋神社是强体力点

“天空鸟居”景色值，但徒步约 40–60 分钟，上段 270 级台阶很硬。公交班次少，自驾停车也可能满；清晨或日落前最美，必须查观音寺市官网确认登山道开放。

观音寺高屋神社防滑鞋

6｜高知西部是隐藏大头

四万十川、足摺岬、大堂海岸、柏岛、牧野公园、竹林寺/牧野植物园等都能强化“清流 + 海岬 + 植物”的疗愈感，但交通分散，只能作为素材池，不要全部塞进 13 天。

Setouchi branch

如果想看艺术岛，把它做成分支，不要硬塞进疗愈主线

截图给出的跳岛经验很明确：6 天可跑紧凑型精选岛屿，7–9 天再增加西向/骑行/鸣门等内容；5 天只能做高松 + 直岛 + 丰岛。

5 天最小可行版

  • D1 高松：栗林公园 / 屋岛 / 北滨三选二。
  • D2 直岛：地中美术馆、南寺、李禹焕/Benesse、黄南瓜；租电助力车。
  • D3 丰岛：丰岛美术馆、心脏音资料馆，控制点位。
  • D4 小豆岛或高松补缺：妖怪美术馆/迷路街道/橄榄公园，或回高松休息。
  • D5 机场返程：只买纪念品和吃早餐。

6–9 天增强版

天数安排关键经验
6 天高松区 2 选 1 → 直岛 → 丰岛 → 小豆岛 → 女木/男木 → 返程紧凑但可行，适合艺术祭航班窗口。
7 天在 6 天基础上补一个高松没去的点 / 小豆岛没去的点 / 瀬户大桥沙弥岛胜在人少，不要继续加远路。
8 天向西：金刀比罗宫 + 四国水族馆，或岛波海道初级骑行建议换住丸龟/坂出/今治，不要天天从高松拉车。
9 天鸣门涡潮可考虑；也可从神户/淡路岛过来远且独立，只有余裕时再放。

小豆岛

推荐抓“酱油、橄榄、港口小镇、海边咖啡”。岛大且公交不密，1 天只选 2–3 个点。

栗林公园

高松最稳的慢启动点。园内喝茶、坐船、找机位都可以，但核心是半天慢节奏。

高松港 / 北滨

跳岛中转与城市恢复点。住港口/北滨/片原町一带，去岛和觅食都方便。

Where to stay

住宿基地怎么选

之前 PDF 的住宿原则是“地方感优先于豪华”。截图进一步补充：跳岛期不要默认一直住高松，旺季/艺术祭时宇野、丸龟、坂出会更有策略价值。

高松

适合首次抵达、栗林公园、高松港、北滨、屋岛、小豆岛；机场巴士进市区方便，夜生活/餐饮更丰富。

建议区域：高松港、北滨、片原町、瓦町。

坑：艺术祭/假期房价高，跳岛去直岛/丰岛未必最省时间。

宇野

适合把直岛/丰岛作为核心的人。离港口近就是巨大加分，船程更短、排队压力更低。

截图民宿：Sadou Inn，约 400/晚，Airbnb 或官网 sadouinn.com；近港口、711、麦当劳、超市和早市。

坑：餐饮和夜生活少，公共浴室/厕所需接受。

丸龟 / 坂出 / 今治

适合 8–9 天西向延伸：四国水族馆、瀬户大桥、金刀比罗宫、岛波海道初级骑行。

策略：不要天天从高松往返拉车；换宿能省体力和交通费。

坑：城市体验比高松弱，先确认晚餐和回酒店交通。

Kochi / Shimanto add-on

高知西部增补：只当素材池，不全塞

高知县西部的自然很强，但分散。最适合加在 D9–D10 的四万十段，或下一次单独做“高知西部 4–5 天”。

优先级最高

  • 四万十川游览船 / 屋形船 / 轻骑行：路线主线已经包含，保留。
  • 佐田沉下桥：四万十最代表性的低强度点，选 1–2 座桥即可。
  • 四万十复古小火车 / JR 予土线：适合不自驾的人补一点铁路风景。
  • 高知市内：竹林寺、牧野植物园、五台山展望台，可作为 D8 或备选雨天城市点。

慎重加入

  • 足摺岬、白山洞门、大堂海岸、柏岛：景色强，但从中村/宿毛继续拉车，容易把疗愈变成赶路。
  • 牧野公园：JR 佐川站步行可达，适合作为高知植物主题加点，但不应挤掉四万十河边留白。
  • 海洋堂 / 打井川：有趣但偏专题，除非本人强兴趣，否则不如把时间还给住宿和河边。
Practical checklist

执行清单：这几件事比多加景点重要

把截图经验转成可执行动作，减少临时踩坑。

交通与票券

  • ALL SHIKOKU Rail Pass 只在最终航班、住宿位置定后再算，重点看 D3–D9 或 D4–D10 是否覆盖高频铁路/渡轮。
  • 日本交通费贵；能接受慢车就少用特急/新干线，Google Maps 先比价再决定。
  • 祖谷、奥祖谷、高屋神社、四万十活动都不能临时查；班次少的地方提前截图时刻表。
  • 如果只有中国大陆驾照，不要默认能在日本租车；出发前按 JAF/租车公司规则核验。

住宿与餐饮

  • 祖谷/大步危 3 晚、四万十 2 晚、大洲/内子 2 晚仍是 PDF 主线的体验核心，优先级高于景点。
  • 跳岛旺季优先抢宇野/高松港/北滨/瓦町；8–9 天西向延伸再考虑丸龟/坂出。
  • 早餐有自助/吧台会省很多心力；乡间晚餐尽量订旅宿餐，别临时找。
  • 可用超市食材自己简单处理，外食选轻服务的大众酒场/定食，降低客单价。

打包与风险

  • 防滑登山/步行鞋：高屋神社、琴平石阶、祖谷藤桥、河边雨后都需要。
  • 轻薄防风外套、雨具、小毛巾、现金、小药包、能量棒/零食。
  • 高屋神社山顶风大，广角镜头可用，但禁止三脚架/无人机。
  • 台风/大雨时删河上活动和深山移动，保留栗林公园、北滨、高知/内子室内与咖啡。

取舍规则

  • 13 天主线：别把直岛/丰岛强塞进去，除非愿意删掉小豆岛或缩短祖谷/四万十。
  • 跳岛线：别追全岛全馆，艺术岛每岛抓 2–3 个主点即可。
  • 低体力时优先删除：金刀比罗宫登顶、奥祖谷、足摺岬/柏岛远征、宇和岛停留。
  • 必须保留：高松落地慢启动、祖谷住宿、四万十河边、大洲/内子收束。
