# 四国旅行地图｜晓倩专属攻略

一个定制化的四国旅行决策网站：用交互式示意地图组织 13 天疗愈主线、四县区域攻略、瀬户内跳岛分支、住宿与交通取舍规则。

## 内容来源

- 四国疗愈旅行攻略 PDF：2026.09.25–10.07，13 天 12 晚主线。
- 旧本地四国专页与四条路线总览。
- 小红书截图经验提炼：栗林公园、直岛/丰岛、小豆岛、高屋神社、高知西部、交通票券与住宿策略。

## 本地运行

```bash
npm install --legacy-peer-deps
npm run dev
```

默认开发地址：<http://127.0.0.1:5184/>

## 构建

```bash
npm run build
npm run preview
```

默认 preview 地址：<http://127.0.0.1:4184/>

## 当前功能

- 交互式四国示意地图：四县 + 10 个地点/分支点。
- 地图点击切换区域攻略与地点攻略。
- “想保留 / 待定 / 可删”本地浏览器决策标记。
- 13 天主线按县筛选。
- 瀬户内跳岛、温泉强化、高知西部等分支策略。
- 桌面与移动端响应式布局。
