# 四国旅行地图｜专属攻略

一个定制化的四国旅行决策网站：用独立放大的交互式四国地图组织 13 天轻松主线、四县景点池、瀬户内跳岛分支、图片图集、住宿与交通取舍规则。

## 内容来源

- 四国疗愈旅行攻略 PDF：2026.09.25–10.07，13 天 12 晚主线。
- 旧本地四国专页与四条路线总览。
- 小红书截图经验提炼：栗林公园、直岛/丰岛、小豆岛、高屋神社、高知西部、交通票券与住宿策略。
- 官方/半官方旅游资料与 Travel Japan/JNTO 资料：补充鸣门漩涡、涡之道、大塚国际美术馆、阿波舞、仁淀川、岛波海道等四县景点。

## 本地运行

```bash
npm install --legacy-peer-deps
npm run dev
```

默认开发地址：<http://127.0.0.1:5184/>

## 构建

```bash
npm run build
npm run preview
```

默认 preview 地址：<http://127.0.0.1:4184/>

## 当前功能

- 单独地图页：地图作为独立大画布展示，不再挤在侧栏里。
- 真实四国县域轮廓地图：香川、德岛、高知、爱媛使用 GeoJSON 轮廓生成。
- 40 个可选景点池点位：香川 9、德岛 12、高知 10、爱媛 9；德岛重点补入鸣门漩涡、大鸣门桥/涡之道、观潮船与周边组合。
- 地图点击切换区域攻略与景点详情，县域景点池和底部总览同步选择。
- 景点图集：50 个点位共 188 张本地参考图，详情页显示大图 + 缩略图，底部景点池也带缩略图。
- “想保留 / 待定 / 可删”本地浏览器决策标记。
- 13 天主线压缩为底部简表，瀬户内跳岛、鸣门观潮、仁淀川、岛波海道等作为可替换分支。
- 桌面与移动端响应式布局。
